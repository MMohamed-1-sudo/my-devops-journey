lab name [ calculator ]

objective: 
create a working calculator which could do simple addition, subtraction, multplication and division. 
using Ifs and elif statment and echo commands. 

commmands used:
"#!/bin/bash

echo "first number: "
read a
echo "operator : ( + * - / )"
read op
echo "second number: "
read b 
 
 if [ "$op" = "-" ]; then 
  echo "$(( a - b ))"
 elif [ "$op" = "+" ]; then
  echo "$((a + b))"
 elif [ "$op" = "*" ]; then
  echo "$((a * b))"
 elif [ "$op" = "/" ]; then
  echo "$((a / b))"
 fi"

OUTPUT:

first number: (user input number)
operator ( + * - / ): (user input operator)
second number: (user input number )
